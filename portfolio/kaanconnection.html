Kaan's Smart Connection

Smart connection was een vak waarbij je leert hoe je daadwerkelijk producten online kan krijgen. Tijdens dit vak leer je dingen te doen zoals het maken van een website met het gebruik van verschillende code talen, integreren van API's, wat thema's zoals data analytics, AI of BI is.

Ik vond de opdrachten van Smart Connection niet altijd de leukste, ik had sowieso al programmeer ervaring. Ik heb in mijn eerste jaar namelijk al codes geschreven voor websites. Maar toch vond ik het tempo erg snel gaan voor leerlingen die dit nog nooit gehad hebben. Zo heb ik meerdere medestudenten geholpen met vragen over een code. Wat ik al een grote minpunt was, was dat je hele grote stukken codes krijgt zonder een toepassende uitleg over wat de codes begrepen. Ik denk dat je daarmee het voor beginnende studenten al te lastig maakt. Ook zaten er in de grote stukken code af en toe fouten, waardoor er fouten optreedde. Een daarvan was de API waarbij er een suf typfoutje zat in de variabele, maar tijdens het programmeren kan dit wel grote gevolgen hebben vooral als je beginnend student bent in dit vak gebied. Voor de rest vond ik de API's, en sites wel leuk om te maken. Ik heb nog niet eerder een API gebruikt en was voor mij een leerzame ervaring waarin ik heb begrepen hoe een API werkt. Daarnaast waren de meeste themas daarna voor mij bekend al vond ik de spambox functie wel een leuke methode om AI te introduceren en formules te gebruiken binnen ICT.
