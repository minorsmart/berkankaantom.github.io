Berkan's Smart Connection

Tijdens Smart Connection kregen wij lessen over programmeren, zoals HTML, CSS en javascript. Wij moesten zelf dus een eigen website bouwen voor ons eigen projectgroep. Ondanks mijn reeds opgebouwde programmeerervaring tijdens mijn studie HBO-ICT was dit geen nieuw terrein voor mij. Desondanks bood deze course een waardevolle kans om mijn vaardigheden verder te ontwikkelen en toe te passen.

Tijdens de lessen werkten we aan diverse activiteiten en kregen we instructies over API, javascript en over AI, zoals bijvoorbeeld ChatGPT. Het praktische aspect van de opdrachten maakte de course dynamisch en boeiend.

Opdrachten tijdens de lessen

Een van de eerste opdrachten betrof het maken van een website waarop de actuele weersverwachting per locatie afleesbaar was, dus een API. Mijn API was gericht op de weersverwachting in Istanbul.

Een tweede opdracht was het ontwikkelen van een webpagina d.m.v. API van NASA die de "Foto van de Dag" van NASA toonde.

Hoewel deze opdrachten geen nieuwe uitdagingen op het gebied van programmeren voor mij vormden, waardeerde ik de gelegenheid om mijn bestaande vaardigheden te verfijnen. De course Smart Connection bood een gestructureerde aanpak om websites te ontwikkelen en tegelijkertijd inzicht te krijgen in het integreren van diverse elementen op een platform. Het was een waardevolle toevoeging aan mijn technische kennis.
